About us!

This web project focuses on managing a automobile shop between owner and his costumers. The automobile shop management system is a system that provides a way to manage automobiles in the shop. This includes a registration of automobile with its different details such as company, model, colour, price, availability, features etc. If the customer wants to buy any automobile from the shop they also need to fill up the transaction and costumer registration form. The owner of the shop manages the orders through the website. And can manage the data about the automobiles.

Through this project we are able to efficiently manage a shop data and reduce any errors in the management and delivery of the automobiles. We are also allowing customers to order their automobiles through the website so they can select their choice efficiently by viewing all their options the shop has to offer. Customers can track their delivery date assigned by the owners or can also schedule a visit to collect their orders. At the end all we want that the workload on both owner and customer side is reduced with the help of technology

The website is designed using HTML and its attributes and styling of the HTML pages is done using CSS. JavaScript is used to make the site more interactive, user friendly and responsive. MYSQL is used as database management system to efficiently store and retrieve information in the database and also to remove data redundancy. It also helps to prevent any false entries in the database using its different methods like foreign key etc. The backend software provides connectivity between the frontend and the database And is responsible for smooth running of the website.
